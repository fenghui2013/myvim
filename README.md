### myvim ide
* python ide
* support GBK

### user guide
```
add alias myvim='vim -S mysession.vim -c "WMToggle"' into .bashrc to save session
commands:
qqq(vim): save session, modifies and quit
myvim(shell): restore the state that is quit last
```
